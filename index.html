Auto-Citizenship - ItalyBA

About

Python bot for booking Italian citizenship reconstruction appointments at the Buenos Aires Consulate.

Requirements:

  • Windows Operating System: Tested on Windows 10
  • Google Chrome: Tested on version 129.0.6668.90 (Official Build - 64 bits)
  • Internet Access: Required to access the appointment booking website

Get it

  • Option 1: Clone
    • git clone https://github.com/Damocod/auto-citizenship-italyba.git
  • Option 2: Download source as ZIP
  • Option 3: Download a portable version

How to use it

Once downloaded or cloned read the README file and follow it's instructions
